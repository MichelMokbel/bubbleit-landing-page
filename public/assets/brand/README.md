Drop the official Bubbleit asset package into this directory with these filenames:

- `logo-primary.svg`
- `logo-primary.png`
- `logo-secondary.svg`
- `logo-secondary.png`
- `logo-white.svg`
- `logo-white.png`
- `brand-pattern-primary.svg`
- `brand-pattern-primary.png`
- `brand-pattern-secondary.svg`
- `brand-pattern-secondary.png`

This implementation ships with SVG fallbacks using the same names that the app already references.
Replace those files directly to switch to the official brand assets without touching the code.
